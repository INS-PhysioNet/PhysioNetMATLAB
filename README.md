# PhysioNet MATLAB Code

Code shared between members of the PhysioNet team at the [Institut de Neurosciences des Systèmes](https://ins-amu.fr/) (Aix-Marseille université, Marseille, France).

## Installation

After downloading the repository, on MATLAB, run:
```
>> addpath(/path/to/PhysioNetMATLAB) % add main folder to MATLAB path, allowing to run next line
>> addpathR(/path/to/PhysioNetMATLAB) % from now on, addpathR can be used to add folders and subfolders
```

## Contents

     Plot          - Advanced plotting.
     Statistics    - Statistical testing.
     Utility       - Convenience functions to simplify script writing.
     Validators    - Argument validation functions.
